### The Dialectical Engine

How five AI agents and a human domain expert used dialectical analysis to debug a neural network — and discovered that the failure record was more valuable than the model.

CLCA / 8D methodology from semiconductor manufacturing applied to neural network development — systematic defect isolation across 16 model versions, each dead end with a documented root cause.

### Sovereign Infrastructure for Citizen Science
*Building a Self-Hosted Research Lab for Radio Propagation Analysis*

How a self-hosted lab processes 14.68 billion rows on commodity hardware — 24.67 Mrps ingestion, 60 GB/s query throughput, 30 tok/s local LLM inference — at roughly 1.3% of equivalent cloud cost.

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.21003263.svg)](https://doi.org/10.5281/zenodo.21003263)

### MCP Beyond AI
*80 Tools, 14 Billion Rows, Zero Cloud — The Full Stack*

The Model Context Protocol isn't an AI thing — it's JSON-RPC over stdio, the integration pattern Unix has used since 1973. What happens when you wire 80 MCP tools, 14 billion rows, a local 70B LLM, and zero cloud into one sovereign stack running from a house in Idaho.

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.21003542.svg)](https://doi.org/10.5281/zenodo.21003542)

The non-technical story of building IONIS — and the framework that governs the agents themselves — told through the lab's "Collected Moments": the caught-and-owned mistakes, where the discipline is that nobody is right and the system doesn't need them to be.
