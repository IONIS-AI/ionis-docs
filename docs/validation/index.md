# Validation Reports

Automated test results from IONIS model evaluations. Each report follows the standardized test report template (see `templates/test_report.md` in the repo root).

## Test Categories

### Polar Tests
Trans-auroral path evaluations verifying Kp sidecar penalizes high-latitude paths correctly.

### Grey Line Tests
Dawn/dusk transition analysis verifying `hour_sin/cos` and `day_night_est` capture rapid SNR swings.

### Heatmaps
Global SNR prediction maps for visual geographic auditing.
